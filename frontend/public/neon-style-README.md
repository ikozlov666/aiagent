# Неоновый стиль иконок и логотипов

Демо: откройте в браузере **`/neon-icons-demo.html`** (при запущенном фронтенде — `http://localhost:5173/neon-icons-demo.html`).

## Как устроен эффект

- **Контур без заливки** — иконки рисуются только `stroke`, `fill="none"`.
- **SVG-фильтр свечения** — в каждом SVG в `<defs>` задаётся фильтр:
  - два раза `feGaussianBlur` с разным `stdDeviation` (2 и 4);
  - `feMerge` склеивает размытия и исходную фигуру — получается «трубка» с ореолом.
- **Цвета** — яркие, контрастные на тёмном фоне:
  - lime `#7fff7f`, cyan `#00e5ff`, magenta `#ff4dd4`, orange `#ffaa33`, pink `#ff5ec4`, white `#fff`.

## Использование в проекте

1. **Отдельные SVG** — скопируйте нужный `<svg>...</svg>` из демо (вместе с его `<defs>`) в компонент или в `public/`.
2. **Логотип** — блок «AI Agent» в демо можно вынести в компонент и подставить свой текст.
3. **CSS** — для кнопок/карточек можно добавить `box-shadow` в тон иконке, например:  
   `box-shadow: 0 0 20px rgba(0,229,255,.4);`

## Добавление новых иконок

Нарисуйте контур в векторном редакторе (одни линии, без заливки), сохраните как SVG, вставьте в разметку и примените к группе фильтр:

```html
<g filter="url(#neon-cyan)" stroke="#00e5ff" stroke-width="2.5" fill="none">
  <!-- пути иконки -->
</g>
```

Фильтр `#neon-*` должен быть описан в том же SVG в `<defs>` (см. примеры в демо).
